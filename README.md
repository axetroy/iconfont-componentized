### Make IconFont Componentized

#### Usage

1. Install CLI

```bash
npm install @axetroy/iconfont-componentized-cli -D
```

2. Add script to your `package.json`

```json
   "script": {
       "gen:iconfont": "iconfont-componentized --url https://example.com"
   }
```

3. Run the script

```bash
npm run gen:iconfont
```
